반응형 UI UX 란?

반응형이란 유저가 홈페이지에 신호를 보내면 컴퓨터가 응답하는 상호호환적 개념
UI(User Interface) 유저가 시각적으로 받아들일 수 있는 것 화면의 구성, 버튼 모양, 색상, 크기, 폰트.. 등 모든 시각적 요소 ( 축약 - 디자인 )
UX(User eXperience) 유저가 직감적으로 기능을 예측할 수 있는 것 ( 기능적인 것, 편함 불편함으로 판단할 수 있는 것)

퍼블리싱 방향

html,css,scss,자바스크립트,제이쿼리,view(프레임워커)

html이란?
hyper text markup language ( 뼈대 )

css
cascading style sheet ( 살 )

java script ( html에 적용 가능 ) 동적인 요소
자바스크립트는 객체 기반의 스크립트 프로그래밍 언어이다.  다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.


j Query
는 html의 클라이언트 사이트 조작을 단순화 하도록 설계된 크로스 플랫폼의 자바스크립트 라이브러리
j query로 로직을 통해 자바스크립트에 적용
